# MANDADEX MVP - SEMANA 1

## OBJETIVO
Sistema inicial funcional para recibir mandados por WhatsApp.

## FUNCIONES
- Formulario de pedido
- Confirmación
- GPS
- Pantalla de pedido en curso
- Envío automático a WhatsApp

## ARCHIVOS
- index.html
- style.css
- app.js

## COMO USAR

1. Abrir index.html

2. Cambiar número WhatsApp en:

app.js

Buscar:

let numero = "5213220000000";

3. Reemplazar por tu número.

Ejemplo:
5213221234567
